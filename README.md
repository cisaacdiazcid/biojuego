## Archivos Principales

- **c2runtime.js**: Contiene la lógica principal del juego.
- **data.js**: Archivo de datos utilizado por el juego.
- **index.html**: Archivo HTML principal que carga el juego.
- **jquery-2.1.1.min.js**: Biblioteca jQuery utilizada en el proyecto.
- **offline.js**: Script para manejar el modo offline del juego.
- **offlineClient.js**: Cliente para manejar el modo offline del juego.
- **sw.js**: Service worker para manejar el cache y el modo offline.

## Instalación

Para instalar las dependencias del proyecto, ejecuta:

```sh
npm install
```



## Ejecución

Para ejecutar el proyecto, abre el archivo `index.html` en tu navegador.
